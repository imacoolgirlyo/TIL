## Data Structure 

자료 구조(Data Structure)는 컴퓨터에서 데이터를 저장하거나 정리하는 방법들이다. 
효율적으로 데이터를 찾고 삽입하고, 삭제하고 하는 조작들을 쉽게 하기 위해서 만들어졌고 대표적인 자료 구조들은 아래와 같다. 

- Queue
- Stack
- Binary Search Tree
- Graph
